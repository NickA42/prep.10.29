Scales

D3 scales is a function that you define with parameters. It is a way for you to input a data value and it then returns a scale output value. Basically the scale function lets you input data so you can get more data. Using various inputs of domains and ranges will help with the visualization of the scales. The main d3scale used is the linear scale and it shows a scatter plot, like all the other ones but the inputs are outputed on a linear trendline proportional to their differences.
